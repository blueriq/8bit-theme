### Text Item
This component displays Text Items.

### Presentation Styles
If you add the presentation style `Succcess` on the Text item, it will be displayed with a green background.
If you add the presentation style `Info` on the Text item, it will be displayed with a blue background.
If you add the presentation style `Warning` on the Text item, it will be displayed with a yellow background.
If you add the presentation style `Error` on the Text item, it will be displayed with a red background.
